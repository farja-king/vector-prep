00 Trace mode

Line art / logos give the cleanest single-colour trace. Color mode posterizes photos into flat vector regions. Greyscale does the same but in tonal shades of grey, no hue.

Load an image, pick a mode above, then Vectorize to run the first trace. After that, every slider/setting below still updates the preview live as you adjust it — Vectorize is only for that first pass (or to force a full retrace on demand).

If a color/shade count looks off (e.g. after switching modes), Refresh re-runs this mode's first-look detection fresh, exactly as if the image had just been loaded — no need to reload the file.

01 Detail

Images are automatically supersampled before tracing for smoother, more accurate curves — these controls fine-tune fidelity on top of that.

Higher = the source is upscaled more before tracing, giving the curve fitter finer pixel data to work from — sharper, more precise curves at the cost of a slower trace and a bigger SVG. Auto picks a sensible value based on image size; raise it manually for maximum precision on a small or detail-heavy source.

02 Delete shapes

Click a shape in the Vector preview to select it (and, at Tolerance scope, every similarly-colored shape near it), then click it again — or press Delete Selected — to remove it. Erases the underlying source pixels, so deletions survive re-tracing. Switches to Vector view automatically.

03 Touch-up brush

Paints directly onto the source pixels (Source view), so edits get traced properly instead of sitting on top as a raster patch. Switches to Source view automatically.

Edits (brush strokes, deletes) are staged on the source pixels immediately — switching to VECTOR view or downloading automatically re-traces first if anything's changed, so there's never a separate step needed.
